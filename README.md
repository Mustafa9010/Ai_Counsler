# Ai_Counsler
# Universal AI Academic & Career Advisor 🎓

A Python-based, command-line Artificial Intelligence assistant that provides personalized academic and career guidance. Powered by Google's Gemini 2.5 Flash model, this tool adapts its advice to the user's specific educational or professional stage—ranging from high school students to tenured professors.

## Features
* **Adaptive AI Persona:** Automatically adjusts the complexity and focus of its advice based on the user's current career/academic level.
* **Interactive Chat Interface:** Terminal-based continuous chat that remembers conversation history.
* **Powered by Gemini 2.5:** Utilizes the latest `google-genai` Python SDK for fast, highly structured, and accurate responses.

## Prerequisites
To run this project, you will need:
* Python 3.x installed on your system.
* A free [Google AI Studio API Key](https://aistudio.google.com/app/apikey).
